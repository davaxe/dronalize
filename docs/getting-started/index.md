# Getting Started

Welcome to the Dronalize getting started guide. This section will help you install the toolbox and get up and running with your first trajectory preprocessing pipeline.

---

## Overview

- **[Installation](installation.md)** — Multiple installation options: PyPI, containers (Docker/Apptainer), Conda, or from source.
- **[Quickstart](quickstart.md)** — Preprocess your first dataset and run a training example in minutes.

---

## Requirements

- **Python** 3.10, 3.11, or 3.12
- **PyTorch** 2.5 or 2.6 (for the ML pipeline)
- **PyTorch Geometric** (for graph-based models)

!!! tip
    If you only need the preprocessing and data utilities, the core `dronalize` package has minimal dependencies and does not require PyTorch.
