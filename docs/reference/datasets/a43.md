# A43

<div class="section-intro" markdown="1">
A43 is a drone-recorded highway trajectory dataset from Germany. It focuses on naturalistic vehicle motion around an off-ramp and congested highway segment, making it a useful reference for lane-level traffic analysis and motion forecasting.
</div>

## Dataset facts

| Field               | Value                            | Notes                                                                                               |
| ------------------- | -------------------------------- | --------------------------------------------------------------------------------------------------- |
| Release year        | 2024                             | Based on the cited dataset paper and release.                                                       |
| Domain              | Highway traffic                  | Recorded on highway A43 near Munster, Germany.                                                      |
| Capture platform    | Drone                            | Overhead aerial recording with a highway-scale field of view.                                       |
| Primary agent types | Cars, trucks, buses, motorcycles | Focused on motorized road users.                                                                    |
| Map context         | Highway corridor                 | Includes lane-level road geometry around the main carriageway and ramps.                            |
| Geographic coverage | Munster, Germany                 | Captures one highway segment with congestion in one direction and ramp activity in both directions. |
| Data format         | CSV trajectory files             | Distributed as tabular trajectory data.                                                             |

## Default processing profile

These are the default Dronalize settings used when processing this dataset.

| Setting | Default |
| ------- | ------- |
| Source sequence | 20 obs / 50 pred @ 10 Hz |
| Effective sequence | 20 obs / 50 pred @ 10 Hz |
| Resampling | None |
| Windowing | 70-frame window, step 25 |
| Filtering | Prune agents with fewer than 2 samples |
| Maps | Full map |

## Version

Dronalize does not currently rely on a stable release version marker for A43. The raw layout and loader paths only expose the recording filenames that the reconstructed map logic uses.

## Normalization

### Agent categories

| Dataset type | Dronalize type | Notes |
| ------------ | -------------- | ----- |
| `Motorcycle` | `MOTORCYCLE` | Direct category mapping from `VehicleCategory`. |

| `Van` | `VAN` | Direct category mapping from `VehicleCategory`. |
| `Bus` | `BUS` | Direct category mapping from `VehicleCategory`. |

### Map types

| Dataset type | Dronalize type | Notes |
| ------------ | -------------- | ----- |
| Reconstructed outer lane marking | `ROAD_BORDER` | Applied to the first and last reconstructed markings in each file-specific lane layout. |
| Reconstructed inner lane marking | `LINE_THIN_DASHED` | Applied to intermediate reconstructed lane markings. |
