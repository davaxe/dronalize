# A43

<div class="section-intro" markdown="1">
A43 is a drone-recorded highway trajectory dataset from Germany. It focuses on naturalistic vehicle motion around an off-ramp and congested highway segment, making it a useful reference for lane-level traffic analysis and motion forecasting.
</div>

## Default processing profile

These are the default Dronalize settings used when processing this dataset.

| Setting | Default |
| ------- | ------- |
| Source sequence | 20 obs / 50 pred @ 10 Hz |
| Effective sequence | 20 obs / 50 pred @ 10 Hz |
| Resampling | None |
| Windowing | 70-frame window, step 25 |
| Screening | Prune agents with fewer than 2 samples |
| Maps | Full map |

## Dataset compatibility

Dronalize targets the release or raw layout below. If you have an older or newer download, expect breakage when split names, file names, schemas, or map assets differ.

| Field | Value |
| ----- | ----- |
| Expected release/layout | A43 CSV recording layout |
| Loader expectation | The loader uses the recording CSV files directly, so changes in filenames or lane-layout assumptions can require adapter updates. |
