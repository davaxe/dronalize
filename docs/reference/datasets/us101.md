# US-101

<div class="section-intro" markdown="1">
US-101 is the other classic NGSIM freeway trajectory benchmark often paired with I-80. It captures lane-level vehicle behavior on US Highway 101 in California and remains a common reference for microscopic traffic modeling.

## Dataset facts

| Field               | Value                        | Notes                                                     |
| ------------------- | ---------------------------- | --------------------------------------------------------- |
| Release year        | 2007                         | Based on the NGSIM US-101 release.                        |
| Domain              | Freeway traffic              | Built around lane-level motion on US Highway 101.         |
| Capture platform    | Roadside traffic observation | Distributed as processed freeway trajectory data.         |
| Primary agent types | Cars, trucks, motorcycles    | Focused on freeway vehicle motion.                        |
| Map context         | Lane-level roadway layout    | Often used for highway forecasting and behavior analysis. |
| Geographic coverage | California, United States    | Covers a single freeway study site.                       |
| Data format         | CSV trajectories             | Released as time-sliced traffic intervals.                |

## Default processing profile

These are the default Dronalize settings used when processing this dataset.

| Setting | Default |
| ------- | ------- |
| Source sequence | 20 obs / 50 pred @ 10 Hz |
| Effective sequence | 20 obs / 50 pred @ 10 Hz |
| Resampling | None |
| Windowing | 70-frame window, step 25 |
| Filtering | Prune agents with fewer than 2 samples |
| Lane-change sampling | Require 3 lane changes; keep 1 in 3 negatives |
| Maps | Full map |
